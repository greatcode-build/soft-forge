# Soft Forge

Soft Forge is a modern marketing landing page built with Next.js, Tailwind CSS, and React. It showcases a software services brand with sections for hero content, service offerings, team profiles, customer testimonials, and contact feedback.

## 📍 Why this project is useful

- Clean, responsive landing page for a software services brand
- Built with Next.js App Router and Tailwind CSS v4
- Includes reusable UI components: header, services, team, contact form, footer
- Supports animated stats, mobile navigation, and rich media assets
- Easy to adapt into a business website, agency portfolio, or SaaS presentation

## 🚀 Key features

- Homepage sections: Home, About, Services, Team, Testimonials, Contact
- Responsive navigation with mobile menu toggle
- Animated counter blocks using `react-countup`
- SVG icon usage with `lucide-react` and `react-icons`
- Static image rendering with Next.js `Image` optimization
- Contact form layout with embedded map section

## ⚙️ Getting started

### Prerequisites

- `Node.js`

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open `http://localhost:3000` in your browser to preview the app.

## 📍 Project structure

- `app/` – main application folder for the Next.js App Router
- `app/page.tsx` – landing page layout and section composition
- `app/layout.tsx` – global layout that wraps all pages
- `app/components/` – reusable UI components for header, services, team, contact, footer
- `app/constants/` – shared data used across multiple components
- `public/` – static assets such as images and icons
